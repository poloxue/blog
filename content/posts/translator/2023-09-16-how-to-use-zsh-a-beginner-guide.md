# zsh 是什么？

zsh，或 Z Shell，是一个 Unix-Like 系统（如 macOS 或 Linux）下的 shell 命令行解释器。它支持强大的自动补全能力，拥有丰富的插件可供选择，具有高可定制性，而且与 bash 充分的兼容。虽然，它看起来比 bash，但是 zsh 却依然是更快。

再者，相比较于 bash，zsh 现在依然在积极的维护中，是一个还在不断成长的项目。
